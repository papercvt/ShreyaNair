Healthcare Analytics

Clinical Pathway Simulation in Peripheral Artery Disease

Team Project (5 members)

Simulated peripheral artery disease patient pathways using Synthea modules and built reusable analytics workflows to evaluate survival outcomes, identify clinical bottlenecks, and measure length of stay associated with wound deterioration.

DataSynthetic patient-level pathway data generated with Synthea
AnalysesSurvival, bottleneck profiling, and LOS for wound deterioration
OutputReusable dashboards and code structure for real-world healthcare data

Overview

This capstone project focused on simulating peripheral artery disease clinical pathways using synthetic healthcare data generated through Synthea modules. The goal was to create an analysis framework that could be reused on real-world healthcare data to evaluate patient outcomes, identify inefficiencies in care pathways, and support operational decision-making.

Project Scope

  • Generated synthetic patient pathway data using Synthea modules
  • Conducted three main analyses: survival analysis, bottleneck profiling, and length-of-stay analysis for wound deterioration
  • Built interactive dashboards to communicate pathway performance and support exploration of results
  • Structured code and outputs to be reusable for future real-world healthcare datasets

My Contributions

  • Analyzed time-to-event patterns and outcome differences across simulated patient pathways
  • Helped translate analytical results into dashboard-ready insights
  • Contributed to the team’s overall framing of the project as a reusable healthcare analytics workflow

Why It Matters

This project shows how simulation and analytics can be combined to study healthcare pathways before deploying on sensitive real-world data. It also demonstrates the ability to build reusable, decision-oriented tools for outcome analysis and operational improvement.
