Breast Ultrasound Image Classification

Team Project (5 members)

Built and evaluated a deep learning pipeline for classifying breast ultrasound images, comparing a baseline CNN against fine-tuned ResNet18 to support medical decision-assistance use cases.

Dataset780 ultrasound images across 3 classes
Best modelFine-tuned ResNet18
ResultAbout 92–94% validation accuracy

Overview

This project examined how transfer learning can improve performance on a small, noisy medical imaging dataset. The central comparison was between a baseline convolutional model and a fine-tuned pretrained network.

What I Did

  • Prepared and standardized ultrasound image inputs for modeling
  • Built a baseline CNN from scratch as a benchmark
  • Fine-tuned ResNet18 for a 3-class image classification task
  • Compared validation accuracy, convergence behavior, and generalization
  • Framed the system as decision support rather than diagnostic replacement
