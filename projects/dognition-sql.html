Analyzing Dognition

Performed exploratory analysis on a confidential database using SQL joins, CTEs, and window functions to uncover customer behavior patterns and generate recommendations to improve retention and customer experience.

Core methodsJOINs, CTEs, window functions
FocusBehavior patterns and retention
OutcomeBusiness-facing recommendations

Overview

This project centered on using SQL as an investigative tool. Rather than simply extracting rows, I used layered query logic to connect customer and behavioral data and identify patterns that could inform business action.

What I Did

  • Wrote multi-table SQL queries to combine customer and behavioral records
  • Used CTEs to structure the analysis into readable, reusable steps
  • Applied window functions to compare customer segments and rankings
  • Interpreted patterns in a business context rather than stopping at query output
  • Developed recommendations aimed at customer experience and retention
