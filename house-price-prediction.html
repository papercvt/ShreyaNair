Machine Learning

Mortgage Risk & Property Valuation Modeling

Team Project (5 members)

Developed a machine learning pipeline to estimate residential property values and support mortgage underwriting decisions by identifying overvalued properties and improving risk-based lending strategies.

DatasetWorked with Ames Housing Dataset (~2,900 records, 70+ features)
ApproachLASSO regression, XGBoost, Random Forest
Use caseValuation accuracy and risk thinking

Overview

Traditional property appraisals can be subjective and inconsistent, leading to potential overvaluation and increased lending risk. This project aimed to build a data-driven valuation model to provide an objective benchmark for fair market value, enabling financial institutions to make more informed mortgage decisions.

What I Did

  • Contributed to data preprocessing, feature engineering, and model development
  • Helped implement and compare multiple machine learning models
  • Supported analysis of model performance and interpretation of results
  • Collaborated on translating model outputs into business insights

Tools

PythonXGBoostRandom ForestRisk Analysis

Why It Matters

This project highlights both technical modeling ability and the ability to frame model performance in terms that matter to real decision-makers.
